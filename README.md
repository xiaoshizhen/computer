# C++ 多功能科学计算器

一个用 C++17 编写的命令行多功能科学计算器，支持交互式表达式输入和 40+ 数学函数。

## ✨ 功能特性

### 🔢 基本运算
- **加 / 减 / 乘 / 除 / 取模** — `+` `-` `*` `/` `%`
- **幂运算** — `^` 支持任意底数和指数（右结合）
- **括号** — `(...)` 支持任意嵌套
- **隐式乘法** — `2pi`, `3(1+2)`, `(2)(3)`, `sin(x)cos(x)`
- **科学记数法** — `1.5e10`, `2E-3`

### 📐 三角函数（支持弧度/角度切换）
- **基本**: `sin(x)` `cos(x)` `tan(x)`
- **反三角**: `asin(x)` `acos(x)` `atan(x)` `atan2(y,x)`
- **双曲**: `sinh(x)` `cosh(x)` `tanh(x)`
- **反双曲**: `asinh(x)` `acosh(x)` `atanh(x)`

### 📊 对数与指数
- `ln(x)` — 自然对数
- `log(x)` — 常用对数（底 10）
- `log2(x)` — 底 2 对数
- `logb(x, base)` — 任意底对数
- `exp(x)` — e^x

### 🔣 数学函数
- **幂/根**: `sqrt(x)` `cbrt(x)`
- **取整**: `floor(x)` `ceil(x)` `round(x)` `trunc(x)`
- **其他**: `abs(x)` `sign(x)` `gamma(x)` `hypot(x,y)`

### 🎲 组合与数论
- `perm(n, r)` — 排列 P(n,r)
- `comb(n, r)` — 组合 C(n,r)
- `gcd(a, b)` — 最大公约数
- `lcm(a, b)` — 最小公倍数
- `isPrime(n)` — 质数判断

### 📈 统计
- 均值、中位数、样本标准差

### 🧮 常量
- `pi` (π), `e`, `phi` (黄金比例 φ), `sqrt2`, `ln2`, `ln10`
- `ans` — 上一次计算结果

### 🎯 特色功能
- **角度模式切换** — 弧度/角度一键切换，三角函数自动适配
- **计算历史** — 自动保存最近 50 条计算记录
- **交互式界面** — 直接输入表达式即时求值
- **传统菜单模式** — 输入 `menu` 切换为菜单引导模式

## 🚀 编译和运行

### 使用 CMake（推荐）

```bash
mkdir build && cd build
cmake ..
cmake --build .
./calculator        # Linux / macOS
calculator.exe      # Windows
```

### 直接编译

```bash
# g++
g++ -std=c++17 -o calculator main.cpp calculator.cpp

# MSVC
cl /EHsc /std:c++17 main.cpp calculator.cpp /Fe:calculator.exe
```

## 📁 项目结构

```
.
├── CMakeLists.txt    # CMake 构建配置
├── calculator.h      # 计算器类声明（API）
├── calculator.cpp    # 计算器实现 + 递归下降表达式解析器
├── main.cpp          # 交互式命令行界面
└── README.md
```

## 🏗️ 架构

- **Calculator 类** — 纯静态方法，所有数学功能封装在一个类中
- **递归下降解析器** — 表达式求值使用递归下降法，支持 4 级运算符优先级
- **角度模式** — 全局状态控制三角函数的输入输出单位
- **隐式乘法** — 解析器自动识别省略的乘号

### 运算符优先级（从低到高）

| 优先级 | 运算符 | 结合性 |
|--------|--------|--------|
| 1 | `+` `-` | 左结合 |
| 2 | `*` `/` `%` | 左结合 |
| 3 | `^` | 右结合 |
| 4 | 一元 `-` `+` / 函数 / 括号 | — |
